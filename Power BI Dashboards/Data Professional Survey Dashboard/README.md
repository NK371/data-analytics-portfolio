# Data-Professional-Survey-Dashboard
Power BI Dashboard for Data Professional Survey

![image](https://github.com/user-attachments/assets/a3c0730c-4064-48b2-90bb-ffe3612c1a43)

## Project Overview

This project analyzes a global **Data Professional Survey dataset** to uncover insights about:

- Career entry difficulty
- Salary distribution across job roles
- Work-life balance & salary satisfaction
- Geographic distribution of respondents
- Programming language preferences

The goal of this project was not just to build a dashboard, but to translate raw survey data into **actionable career and industry insights** using Power BI.

This project demonstrates:

- Data cleaning & transformation
- KPI design
- Data modeling
- Insight generation
- Dashboard storytelling

---

## Business Problem

The data industry is growing rapidly, but aspiring professionals often lack clarity about:

- Which roles pay more?
- How difficult is it to break into data careers?
- What skills are most popular?
- How satisfied are professionals with salary and work-life balance?

This dashboard answers these questions using survey-based data.

---

## Dataset Description

The dataset contains responses from **630 survey participants** across multiple countries and job roles.

Key fields include:

- Country
- Age
- Job Title
- Average Salary
- Difficulty in Entering Data Profession
- Work/Life Balance Rating (0–10)
- Salary Satisfaction Rating (0–10)
- Favorite Programming Language

## Dashboard Breakdown & Insights

### High-Level KPIs

- **Total Survey Takers:** 630
- **Average Age:** 29.87 years

Insight:  
The data professional community skews relatively young, indicating that the industry is still expanding and attracting early-career talent.

---

### Country Distribution

Countries include:

- United States
- India
- Canada
- United Kingdom
- Other

Insight:  
The largest respondent group comes from the United States, followed by India. This aligns with global trends where data roles are heavily concentrated in tech-driven economies.

This geographic segmentation enables understanding of global workforce distribution in data roles.

---

### Difficulty in Entering Data Profession

Breakdown of responses:

- Neither Easy Nor Difficult (42.7%)
- Difficult (24.76%)
- Easy (21.27%)
- Very Difficult (6.98%)
- Very Easy (4.29%)

Insight:  
Nearly half of respondents perceive entering the data field as neutral. However, over 30% find it difficult or very difficult, indicating:

- High competition
- Skill barriers
- Possibly unclear entry pathways

This suggests an opportunity for structured career guidance and accessible learning pathways.

---

### Happiness with Work-Life Balance

Average Rating: **5.74 / 10**

Insight:  
Work-life balance appears moderate. While data careers are often seen as flexible, the mid-range score suggests:

- Possible workload pressure
- Deadlines and delivery expectations
- Performance-driven environments

---

### Happiness with Salary

Average Rating: **4.27 / 10**

Insight:  
Despite competitive salary perception in data roles, satisfaction appears relatively low.

This gap between actual salary levels and satisfaction could indicate:

- High expectations
- Cost of living differences
- Comparison-driven dissatisfaction
- Role-based pay disparity

This insight adds depth beyond simply analyzing average salary.

---

### Average Salary by Job Title

Roles analyzed:

- Data Scientist (Highest)
- Data Engineer
- Data Architect
- Data Analyst
- Database Developer
- Student/Looking/None

Key Insights:

- **Data Scientists earn the highest average salary**, reflecting advanced modeling and machine learning expertise.
- Data Engineers and Architects follow closely.
- Students and early-career professionals earn significantly less.
- There is a clear salary hierarchy aligned with technical depth and specialization.

This reflects industry demand for advanced analytical and engineering skills.

---

### Favorite Programming Language

Languages analyzed:

- Python (Dominant)
- R
- Other
- C/C++
- JavaScript
- Java

Insight:

- **Python is overwhelmingly dominant**, confirming its industry-standard status in data science and analytics.
- R maintains niche usage.
- Traditional languages (Java, C/C++) show lower preference among survey participants.

This validates Python as the most strategic language for aspiring professionals.

---

## Key Strategic Insights

1. The data industry is young and growing.
2. Entry barriers exist but are not overwhelming.
3. Python is the dominant skill.
4. Salary increases significantly with specialization.
5. Work-life balance is moderate.
6. Salary satisfaction is lower than expected relative to role pay.
